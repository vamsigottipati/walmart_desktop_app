About OpenRouter

Your API key is stored only in the app's local settings file in your user data directory and is never sent to the renderer or written to browser storage. When a key is present, enrichment uses deepseek/deepseek-v4-flash to synthesize the final profile. If the key is missing or the call fails, the app falls back to local heuristic aggregation.
